Weather App

A weather app created so that I could practice working with simple JSON data as well as the canvas element. When there is precipitation, the canvas will draw rain or snow and animate in the background.

Development with React.js and the Meteor framework allows it to be deployed both to the web as well as mobile app stores.

E-Commerce

After seeing a lot of e-commerce interactions on Dribble, I thought I would try my luck at recreating some with a simple ecommerce site of my own. On mobile, the app feels near native. It is performant and fun to use.

I used react for the front end, with node.js, express, and moltin to serve static files, fetch products, and retrieve related data

Movie Fig

A simple Movie Showtime app to practice working with JSON data. Although the data is accurate when searching, I was unable to gain permission to use the API's movie poster images.

Development with React.js and the Meteor framework allows it to be deployed to both the web and various app stores.

Christmas

For Christmas this year I decided to do something a little different to express some holiday cheer to my loving family. Instead of the typical hallmark cards we purchase at our local stores, I created a little web app for them.

Each of my family members received a password christmas morning as well as a link to the app. Upon arriving, each of them can click on his or her name and enter the password received that morning. Upon entering they are animated to their christmas card with a loving message.

Dog Finder

I came across petfinder's api when searching for data to use in one of my first node.js apps. I wanted to make a simple CRUD capable app that was not one of the many "Todo apps" that are in tutorials everywhere.

The app I made allows a user to search for a dog by any combination of breed, age, size, gender, and location. The user can save the dogs they like to his or her "cart" and compare them side by side or view them later. Built with React, Node, Express, and MongoDB.

Eat Better

Yelp has been a reliable tool in deciding where I'll go for a meal or to enjoy some nightlife for as long as i can remember. This app utilizes their api in conjunction with a fun to use UI.

The primary purpose of creating this app was to up my game when designing in the browser. All animations are done with CSS3 and vanilla javascript. It has a React front end with a Node.js/Express.js backend.

Clean Time

A calculator app for sobriety time. With inspiration from the Alcoholics and Narcotics Anonymous programs, the logo, your chip, and the menu icon change to the color of the users AA/NA Token. Keep coming back!

Development with React.js and the Meteor framework allows for deployment to both the web and various app stores.
